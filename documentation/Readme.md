# **_SarsatJRX_** Beacon Decoder documentation
This is the documentation of F4JRX's SarsatJRX Beacon Decoder.

## Description
SarsatJRX is a COSBAS/SARSAT 406 MHz Beacon Decoder.

It's been developed by F4JRX, emergency radio operator from french civil security association [ADRASEC 31](https://adrasec31.canalblog.com/).
SarsatJRX is meant to be used on the ground to help locating emergency beacons from [COSPAS/SARSAT](https://www.cospas-sarsat.int/) system.

Once connected to an UHF FM receiver tuned to the beacon frequency, SarsatJRX will decode the 406 frames transmitted by the beacon and show its GPS coordinates.
Coordinates can sent to a phone for direction finding, either by scanning a QR Code on SarsatJRX's screen, or by connecting the [[dedicated app|SarsatJRX App]] to the decoder.
